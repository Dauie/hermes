
### Most Valuable TCP "Ping" Ports
	- 80 (https)
	- 25 (smtp)
	- 22 (ssh)
	- 443 (https)
	- 21 (ftp)
	- 113 (auth)
	- 23 (telnet)
	- 53 (domain)
	- 554 (rtsp)
	- 3389 (ms-term-server)
	- 1723 (pptp)

### Window Scan
	- Similar to ACK, however, when you send a RST back from an open port_stats
	the Window value will be positive, if the Window is set to zero, the port_stats
	is actually closed.
